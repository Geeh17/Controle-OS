## Stack utilizada

**Front-end:** React, TypeScipt, TailwindCSS, Next_Js

**Back-end:** Node, Next_Js

**Autenticação:** Next-auth

**Banco de Dados:** Prisma, MongoDB

**Teste:** Jest_js
